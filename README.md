## 시작하기

### 이 프로젝트는 pnpm을 사용합니다.

```bash
curl -fsSL https://get.pnpm.io/install.sh | sh -
```

```bash
# 인스톨
pnpm i
# dev 서버 실행
pnpm dev
```

dev 서버는 3066 포트를 사용합니다. [http://localhost:3066](http://localhost:3066)
